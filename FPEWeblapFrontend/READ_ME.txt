Frontend elindítása:

Töltsétek le a Node-t.

Node version: v18.16.0

Futtasátok végig.

Utána Visual Studio Code-ban nyissátok meg a FPEWeblapFrontend mappát.

Nyissatok egy terminált. Majd írjátok be hogy npm i vagy npm install. Ha hiba nélkül végig futott akkor írjátok be hogy npm run serve és elvileg ez után már működik is a dolog.
